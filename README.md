Requirements:

1. nodejs
2. bower
3. grunt


		$ git clone https://github.com/iionut/git-presentation.git
		$ cd git-presentation
		$ npm install
		$ bower install
		$ grunt serve

Go to [http://localhost:8000](http://localhost:8000)

Presentation build using [Reveal.js](https://github.com/hakimel/reveal.js/)
